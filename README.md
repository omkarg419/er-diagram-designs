# ER Diagram Designs

This repository contains ER diagram designs for different database systems.

## Projects

- Instagram Thrift Store Database
- Fitness Influencer Coaching Platform
- Clinic Appointment and Diagnostics Platform
- Comic-Con Parking System

Each project includes:
- Entities and Attributes
- Primary and Foreign Keys
- Relationships and Cardinality
- Junction Tables (for M:N relations)

Tools Used:
- Eraser
